A virtual reality table tennis trainer developed in Unity. The project focuses on physical fidelity — the ball follows realistic parabolic trajectories with spin influence, and paddle interaction uses collision geometry calibrations, and carefully set physics parameters to match the feel of a real table tennis experience.

The trainer game consists of essentially a "batting cage" but for table tennis, with three varying levels of difficulty, you have to keep up with the opponent AI in responding to each of the serves. The table and environment are modelled to regulation dimensions so spatial intuition from real play transfers to the VR experience.

Hand-tracking input maps controller position and velocity directly to paddle motion each frame, keeping latency as low as possible for responsive feel. The project was packaged and tested as a standalone Unity build and tested with the meta quest 3 at synchron'25
